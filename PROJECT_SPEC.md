# Ecommerce Fullstack Project

Goal

Build a full stack ecommerce application with authentication, product listing, cart management, order management and user profiles.

Users

- Buyer
- Seller
- Admin (future)

Core Features

1. Authentication
   - Signup
   - Login
   - Access token
   - Refresh token

2. Products
   - Product listing
   - Product detail page
   - Featured products

3. Cart
   - Add to cart
   - Update cart
   - Delete cart items

4. Orders
   - Checkout
   - Place order
   - Order history

5. Profile
   - User profile
   - Address management
   - Logout

Frontend Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zod
- Tanstack Query

Backend Stack

- Node.js
- Express
- TypeScript
- Prisma ORM
- PostgreSQL
- Zod

Authentication

- JWT Access Token
- JWT Refresh Token
- Refresh token stored in database

Database

PostgreSQL

MVP Pages

1. Landing Page
2. Product Listing Page
3. Product Details Page
4. Cart Page
5. Checkout Page
6. Order History
7. Profile Page
8. Authentication Pages
9. Featured Carousel
